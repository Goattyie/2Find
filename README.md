# 2Find
*****

**2Find** - игра для двух участников в которой игроки управляют каждый своим персонажем на общей для обоих игроков 2D карте с видом сверху

**Размер карты:** 8000х8000

**Цель игроков** - встретить друг друга

**Условие поражения** - один из игроков встретил вражеского персонажа
*****

Враги начинают двигаться за персонажами игроков когда замечают их на карте и если игроки выйдут из их радиуса обнаружения через некоторое время прекращают погоню

В игре имеется несколько видов врагов:
1. Призрак(может ходить через стены)
2. Ассасин(исчезает и появляется на следующей клетке)
*****

Появление персонажей игроков а также вражеских персонажей происходит случайно
*****

**Виды текстур:**
1. текстуры крыш
2. текстуры дорог
3. текстуры стен
4. текстуры вражеских персонажей
5. текстуры персонажей игроков
